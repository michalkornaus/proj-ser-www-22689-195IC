# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

##### Widok katalogu po utworzeniu projektu przy pomocy polecenia create react app.
![screen1](/screens/1.PNG)
##### Podgląd konsoli podczas uruchomienia lokalnego serwera przy użyciu polecenia npm start. Jak widać możemy dostać się do niego poprzez adres localhost na porcie 3000.
![screen2](/screens/2.PNG)
##### Po wejściu na stronę widzimy domyślny widok projektu który tworzy się automatycznie po utworzeniu projektu. Do mojej aplikacji zostawię szatę graficzną oraz obracające się logo React'a.
![screen3](/screens/3.PNG)
##### Przy użyciu komponentów oraz ich state'ów oraz props utworzyłem prostą aplikację wyświetlającą nazwę sklepu, ulicę oraz przedmioty znajdują się w nim. Pod listą znajdują się dwa przyciski.
![screen4](/screens/4.PNG)
##### Przycisk z lewej *Szczegóły* służy do dodania tekstu do głównego nagłówka z tekstem *Części komputerowe*.
![screen5](/screens/5.PNG)
##### Przycisk po prawem *Zmień kategorie* pozwala łatwo zmienić kategorię przedmiotów które się wyświetlają. Jedną kategorią są podzespoły komputerowe, drugą jest sprzęt dla graczy.
![screen6](/screens/6.PNG)
